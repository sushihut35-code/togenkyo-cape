# InstagramでHPを更新する方法

ホームページの「最新情報」は、**Instagramに投稿するだけで自動的に更新**されます！

---

## 更新方法（超カンタン！）

### 手順

1. Instagramアプリを開く
2. 写真を選んでアップロード
3. キャプション（文章）を書く
4. 「シェア」ボタンを押す

**これだけ！**

↓ 自動的にHPの「最新情報」に表示されます 🎉

---

## 例

### Instagramに投稿する内容

```
📸 写真: アジサイの写真
💬 キャプション:
今日はアジサイが見頃です！🌸
色とりどりのアジサイを楽しんでください。
営業時間は8:00〜17:00です。
```

### HPに表示される内容

```
【写真が表示されます】

今日はアジサイが見頃です！🌸
色とりどりのアジサイを楽しんでください。
営業時間は8:00〜17:00です。

2026年6月10日
```

## よくある質問

### Q: 毎回パソコンが必要ですか？
**A:** いいえ！スマホからInstagramで投稿するだけでOKです。

### Q: 何を投稿すればいいですか？
**A:**
- お知らせ（営業時間の変更、イベントなど）
- 写真（最新のアジサイ、四季の花など）
- 案内（アクセス、駐車場など）

### Q: 文字の長さは？
**A:** キャプションの最初の150文字が表示されます。

### Q: 写真は何枚まで表示されますか？
**A:** 最新の投稿9枚まで表示されます。

### Q: 古い投稿はどうなりますか？
**A:** 新しい投稿が増えると、古い投稿は表示されなくなります（Instagramには残ります）。

## 注意点

### Instagramアカウントについて
- **ビジネスアカウント**にする必要があります
- **公開アカウント**である必要があります
- プライベートアカウントは表示されません

### アカウントの切り替え方
1. Instagramアプリを開く
2. プロフィール右上の「≡」をタップ
3. 「設定とプライバシー」
4. 「アカウントタイプとツール」
5. 「プロフェッショナルアカウントに切り替え」
