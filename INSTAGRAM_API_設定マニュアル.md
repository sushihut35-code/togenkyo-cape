# Instagram API 設定マニュアル

## 概要

このホームページでは、Instagramの投稿を自動的に取得して「最新情報」セクションに表示する機能を実装しています。

**お客様がやること:**
- Instagramに写真と文章を投稿するだけ

**自動的にされること:**
- 投稿がHPに表示される

---

## Instagram API 設定手順

### ステップ1: Instagramアカウントをビジネスアカウントに切り替え

1. Instagramアプリを開く
2. プロフィール画面右上の「≡」メニューをタップ
3. 「設定とプライバシー」→「アカウントタイプとツール」
4. 「プロフェッショナルアカウントに切り替え」を選択
5. 「クリエイター」または「ビジネス」を選択
6. 案内に従って設定を完了

---

### ステップ2: Facebook Developerアカウントを作成

1. [Facebook Developers](https://developers.facebook.com/) にアクセス
2. 「スタートする」をクリック
3. Facebookアカウントでログイン（ない場合は新規登録）
4. 開発者規約に同意してアカウント作成

---

### ステップ3: Facebookアプリを作成

1. Facebook Developerダッシュボードにアクセス
2. 「アプリを作成」をクリック
3. アプリタイプを選択:
   - **「ビジネス」を選択**
   - **「ビジネス管理」**を選択（これが一番簡単）
4. 必要事項を入力:
   - 表示名: `Togenkyo Cape Instagram Feed`
   - アプリ連絡先のメール: ご自身のメールアドレス
5. 「アプリを作成」をクリック

---

### ステップ4: Instagram Basic Display APIを有効化

1. 作成したアプリのダッシュボードに移動
2. 左側のメニューから「製品を追加」をクリック
3. 「Instagram Basic Display」を探して「設定」をクリック
4. 「アプリを設定」をクリック
5. 有効化ドメインを入力:
   - **NetlifyのURL:** `https://enchanting-pothos-073fe4.netlify.app/`
   - **ローカル開発:** `http://localhost:8501`
6. 「保存」をクリック
7. 「ユーザートークン生成」セクションの「または、Instagramユーザーアクセストークンを生成」をクリック

---

### ステップ5: Instagramユーザーアクセストークンを生成

1. 「Instagramユーザーアクセストークンを生成」をクリック
2. Instagramにログイン（ビジネスアカウント）
3. アプリへのアクセスを許可する（「続行」をクリック）
4. 表示されたアクセストークンをコピー

**注意:** アクセストークンは`IGQW...`から始まる長い文字列です。

---

### ステップ6: アクセストークンをscript.jsに設定

1. `script.js`ファイルをテキストエディタで開く
2. 以下の行を探す:
   ```javascript
   const INSTAGRAM_ACCESS_TOKEN = 'YOUR_ACCESS_TOKEN_HERE';
   ```
3. `'YOUR_ACCESS_TOKEN_HERE'` の部分を、ステップ5でコピーしたトークンに置き換え
4. ファイルを保存

**例:**
```javascript
const INSTAGRAM_ACCESS_TOKEN = 'IGQWwL...（実際のトークン）...';
```

---

### ステップ7: 動作確認

1. ブラウザでHPを開く
2. 「最新情報」セクションを確認
3. Instagramの最新投稿が表示されているか確認

---

## アクセストークンの有効期限と更新

### 有効期限
- **60日間**

### 期限が切れるとどうなるか？
- Instagramフィードが表示されなくなる
- エラーメッセージが表示される

### トークンを更新する方法

#### 方法1: 手動更新（推奨）
1. Facebook Developerダッシュボードにアクセス
2. アプリを選択
3. 左側メニュー「Instagram Basic Display」→「ユーザートークン生成」
4. 新しいトークンを生成して、script.jsに貼り付け

#### 方法2: 自動更新ツール
- GitHub Actionsなどの自動更新ツールを設定可能
- これには追加の設定が必要

---

## 注意事項

### セキュリティ
- アクセストークンは**絶対に公開しないでください**
- GitHubなどの公開リポジトリには**コミットしないでください**
- トークンが漏れると、誰でも投稿を取得できてしまいます

### 取得できる投稿数
- **最大20件**（Instagram Basic Display APIの制限）

### APIの制限
- **更新頻度:** 数時間に1回程度が推奨
- 過剰なリクエストは一時的にブロックされる可能性あり

## トラブルシューティング

### 「Instagram連携の準備中です」と表示される場合
- script.jsにアクセストークンが設定されていない
- ステップ6を確認してください

### 「Instagramフィードの読み込みエラー」と表示される場合
- アクセストークンが間違っている
- アクセストークンの有効期限が切れている
- Instagramアカウントがビジネスアカウントになっていない

### 投稿が表示されない場合
- Instagramに投稿がない
- アカウントが非公開になっている（公開アカウントにする必要あり）
- APIの制限に引っかかっている

## お客様向け説明

### Instagramで投稿するだけでOK！

```
1. Instagramアプリを開く
2. 写真をアップロード
3. キャプション（文章）を書く
4. 投稿ボタンを押す

↓ 自動的にHPの「最新情報」に表示されます
```

### よくある質問

**Q: 毎回トークンを更新する必要がある？**
A: 60日ごとに更新が必要です。更新は開発者側（私）が行います。

**Q: 写真はどれくらい表示される？**
A: 最新の投稿が最大9件まで表示されます。

**Q: 古い投稿はどうなる？**
A: 新しい投稿が増えると、古い投稿は表示されなくなります。

**Q: Instagramのアカウントを変えたい場合は？**
A: 開発者側で新しいアカウントのトークンを取得し直します。
